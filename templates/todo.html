Первоочередные задачи

  1. Добавить возможность вместо копирования условия загружать файл с условием.
  2. Добавить возможность загружать файл с разбором в контест.
  3. Добавить возможность по задаче смотреть список контестов, в котором она есть.
  4. Заменить хранение условий в базе на обычное файловое хранилище, а в базе хранить только ссылки на файлы.
  5. Хранить в связи контест-задача также номер задачи в контесте

Задачи второй очереди

  1. Автоматически понимать, какой файл мне загрузили, и, если возможно, компилировать tex для поддержки polygon-like условий.
  2. Заменить выбор тега и задачи из выпадающего списка на что-то более удобное
  3. Ввести группы контестов. Группы должны уметь объединятся с другими группами и контестами, чтобы было удобно хранить ЛОШ-контесты, а также всякие петрозаводские сборы и опентрейнсы.
  4. Сейчас не для всех задач верно, что если у них есть тег, то есть также все предки этого тега. Это произошло потому что я добавил этот функционал несколько позже. Нужно исправить. Исправлять до устаканивания дерева тегов не имеет смысла, поэтому отложил на второй план.

  6. Перейти с просто javascript на typescript или что-то подобное
